# Resumer UI

A UI for the Resumer API created with React, Redux.

## Created with:

- React
- Redux
- Axios
  - AJAX requests to the Resumer API
- React Router
  - Client side routing
- jwt-decode
  - Decoding JSON Web Tokens
- [UIKit](https://getuikit.com/)
  - UI framework
- [SpinKit](http://tobiasahlin.com/spinkit/)
  - Simple CSS spinners
